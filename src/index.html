¿Quiénes somos?

Somos dos estudiantes de Laboratoria interesadas en aprender a programar en Java Script para desarrollar páginas web. Nos interesan los proyectos sociales, ya que estamos convencidas de que la tecnología puede contribuir a generar impactos positivos en la sociedad.

¿Qué hacemos?

Desarrollamos páginas web para formar nuevas maneras de interactuar con la sociedad.
